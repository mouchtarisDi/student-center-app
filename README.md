# 🧠 [Therapy Center Manager](https://student-center-app.onrender.com)


> Turning paper-based therapy centers into structured digital systems.

A full backend-driven web application built to manage real-world therapy centers.

This project was designed, structured, and implemented end-to-end — from database schema to deployment.

---

## ✨ Why I Built This

A therapy center was managing:
- Sessions on paper
- Payments manually
- Evaluation renewals separately
- Student data without centralized structure

I built this system to:
- Automate workflows
- Reduce human error
- Centralize data
- Provide structured tracking
- Improve operational transparency

---

## 🚀 What It Does

### 👨‍🎓 Student Management
- Create & edit students
- Parent contact tracking
- Evaluation expiration monitoring

### 🗓️ Sessions System
- Real sessions with date & time
- Monthly predefined sessions
- Automatic session reset on renewal

### 💳 Payment Tracking
- Add & edit payments
- Full payment history
- Linked to students

### 🏥 Evaluation Renewal Logic
- Track expiration dates
- Option to modify session counts per service
- Restore session availability automatically

### 📝 Comments System
- Add/edit student notes
- Track history
- Structured observation logging
